# Thư mục ảnh Trang chủ (Homepage Images)

Thư mục này dùng để lưu trữ các hình ảnh hiển thị trên Trang chủ. Bạn hãy copy ảnh của mình vào đây và đổi tên tương ứng:

1. **Ảnh đại diện cá nhân (Profile / Avatar):**
   - Tên file gợi ý: `avatar.jpg` hoặc `avatar.png`
   - Kích thước gợi ý: Ảnh vuông, khoảng `800x800px` trở lên để đảm bảo độ nét.

2. **Ảnh nền/banner của phần giới thiệu đầu trang (Hero Banner - nếu cần dùng thay cho dải màu mặc định):**
   - Tên file gợi ý: `hero-bg.jpg`

3. **Ảnh đại diện cho các Thành tựu nổi bật hiển thị ở Trang chủ:**
   - Ảnh kết nạp Đảng viên: `party-member.jpg`
   - Ảnh giải nhất cuộc thi MOS: `mos-champion.jpg`
   - Ảnh giải nhất Ứng dụng số trong dạy học: `digital-teaching.jpg`
   - Ảnh nhận giải thưởng Sao Tháng Giêng: `sao-thang-gieng.jpg`

4. **Ảnh đại diện cho các Đề tài nghiên cứu hiển thị ở Trang chủ:**
   - Ảnh nghiên cứu AI Bot dạy học: `research-ai-bot.jpg`
   - Ảnh nghiên cứu hệ thống E-learning MOODLE: `research-moodle.jpg`
   - Ảnh hội nghị khoa học: `scientific-conference.jpg`
